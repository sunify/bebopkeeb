# Автоматизация с Shortcuts

На маке есть такой апп Shortcuts (по-русски «Команды»), который позволяет выполнять последовательность действий по разным триггерам: таймеры, какие-то системные события, нажатия клавиш.

Это инструкция для клавиатуры, поэтому нас интересуют клавиши. Ниже будет пример настройки.

## Быстрая команда вкл/выкл фокусирование

1. Открываем приложение и нажимаем плюсик
    <img width="972" height="521" alt="изображение" src="https://github.com/user-attachments/assets/0e7d67d0-e5d4-4aee-81b1-64c07aab83d6" />
2. Справа выбираем «Вкл/выкл фокусирование» и переносим в рабочую область
    <img width="1211" height="377" alt="изображение" src="https://github.com/user-attachments/assets/c9c4c552-70f1-4626-91d9-7504db924e4a" />

3. Там же справа вверху нажимаем иконку I и потом кнопку «Добавить сочетание клавиш»
   <img width="396" height="396" alt="изображение" src="https://github.com/user-attachments/assets/dec9d3c4-888b-4688-89c3-e2f37ca035e6" />
4. В появившемся поле ввода нажимаем нужную комбинацию клавиш. Чтобы использовать его с bebop_keeb нужно чтобы на клавишу был назначен макрос с этим сочетанием. Это можно сделать через [VIA](via.md)
   <img width="357" height="205" alt="изображение" src="https://github.com/user-attachments/assets/bb454040-99f3-44ef-95aa-e4cdaf442da8" />
5. Все, по сочетанию, или по нажатию на клавишу мини-клавиатуры, будет вызываться эта команда

## Неприятная часть aka ограничения

Некоторые приложения типа Телеграма, разных аппов на электроне (луп, слак), перехватывают эти хоткеи и не дают службам срабатывать. Т.е. команда может не сработать и нужно в голове держать где будет работать, а где нет.
