# Настройка через VIA

VIA — популярный конфигуратор QMK-совместимых клавиатур, прямо как эта.

## Начало

1. Зайти на сайт https://www.usevia.app/
2. Зайти на вкладку Design
3. На строке Load draft definition нажать «Load»
4. Выбрать файл via.json из корня этого репозитория
5. Тут же появился окно добавления клавиатуры. Можно выбрать здесь или идти дальше
6. Перейти во вкладку Configure

## Возможности

VIA позволяет назначать на любую клавишу любое нажатие: обычные клавиши, модификаторы, медиа-клавиши (плэй/пауза,громкость и т.д.).

Так же можно назначать макросы — сочетания или последовательность нажатий. Сначала макрос нужно записать — это делается в секции MACROS, доступны 15 слотов и потом нзначить один из макросов на клавишу. Так можно по нажатию на клавишу отправлять какой-нибудь хоткей и реагировать на него в системе.
