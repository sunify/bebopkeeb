# Настройка через VIA

VIA — популярный конфигуратор QMK-совместимых клавиатур, прямо как эта.

## Начало

1. Зайти на сайт https://www.usevia.app/
2. Зайти на вкладку Design
3. На строке Load draft definition нажать «Load»
4. Выбрать файл keyboard.json из корня этого репозитория
5. Перейти во вкладку Configure

## Настройка

1. Нажать Authorize device
2. Выбрать bebop_keeb
3. Выбрать нужную клавишу и назначить ей нужное нажатие
4. Готово

## Возможности

VIA позволяет назначать на любую клавишу любое нажатие: обычные клавиши, модификаторы, медиа-клавиши (плэй/пауза,громкость и т.д.). Так же можно назначать макросы — сочетания или последовательность нажатий. Сначала макрос нужно записать — это делается в секции MACRO, доступны 12 слотов и потом нзначить один из макросов на клавишу. Так можно по нажатию на клавишу отправлять какой-нибудь хоткей и реагировать на него в системе.
